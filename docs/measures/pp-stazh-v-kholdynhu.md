### DAX

```dax
VAR _maxd = [PP.history_position_maxd]
VAR _seniority = 
    CALCULATE(
        SELECTEDVALUE(fact_Employee_History_Position[seniority_LAST_HOLDING_HIRE_DATE]),
        'fact_Employee_History_Position'[PERIOD] = _maxd
    )
VAR _years = ROUNDDOWN(_seniority/12,0)
VAR _month = _seniority - _years *12
VAR _res = 
	IF(
		NOT ISBLANK( _seniority ),
		IF(
			NOT ISBLANK( _years ),
			_years & " р."
		) & " " &
		IF(
			NOT ISBLANK( _month ) && _month <> 0,
			_month & " міс."
		)
	)
RETURN 
	COALESCE(
		TRIM(_res),
		"-"
	)
```

**Бізнес-назва:** Стаж в Холдингу

### Опис із ТЗ

Значення поля в місяцях потрібно перевести в роки та місяці. Наприклад, якшо `seniority_FIRST_HOLDING_HIRE_DATE` = 17, то в звіті треба відобразити 1 рік 5 місяців.

Значення поля в місяцях потрібно перевести в роки та місяці. Наприклад, якшо `seniority_LAST_HOLDING_HIRE_DATE`= 17, то в звіті треба відобразити 1 рік 5 місяців.

??? note "Поля-джерела та пов'язані бізнес-метрики (7)"
    | Поле | Бізнес-метрики |
    |---|---|
    | `PERIOD` | Дата нарахування премії Зірка МХП · Дата · Період нарахування · Період |
    | `seniority_LAST_HOLDING_HIRE_DATE` | Стаж в холдингу останній · Стаж в холдингу · Стаж в холдингу (останній) |
