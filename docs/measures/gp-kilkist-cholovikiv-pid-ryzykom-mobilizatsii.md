### DAX

```dax
VAR _admin = 
	CALCULATE(
		COUNTROWS(VALUES('fact_Employee_List'[EMPLOYEE_ID])),
		'fact_Employee_List'[IS_AT_RISK] = 1,
		'dim_Person'[GENDER] = "ЧОЛОВІКИ"
	)
VAR _admin_lt = 
	CALCULATE(
		CALCULATE(
			COUNTROWS(VALUES('fact_Employee_List'[EMPLOYEE_ID])),
			'fact_Employee_List'[IS_AT_RISK] = 1,
			'dim_Person'[GENDER] = "ЧОЛОВІКИ"
		),
		TREATAS( VALUES( 'dim_Admin_LT_OS'[USER_ACCESS_ID] ), fact_Employee_List[USER_ACCESS_ID] )
	) 
VAR _res = 
	SWITCH(
		SELECTEDVALUE('t_HierarchyTypes'[Index]),
		0, _admin_lt,
		1, _admin
	)
RETURN COALESCE(_res, 0)
```

## Бізнес-суть

### Опис із ТЗ

1 - під ризиком. 0 - ризик відсутній

Якщо 1 - Так, якщо 0 - Ні.

Розрахункове поле. Відношення кількості чоловіків під ризиком мобілізації, `is_at_risk` = 1 до загальної кількості чоловіків у команді
